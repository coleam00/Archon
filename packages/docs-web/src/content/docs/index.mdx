## 바로 시작하기

:::code-group

```bash [문서 사이트 개발]
bun install
bun run dev:docs
```

```bash [전체 앱 개발]
bun install
bun run dev
```

```bash [포크 동기화]
git fetch upstream
git switch dev
git merge upstream/dev
```

```bash [작업 브랜치 시작]
git switch -c ko/landing-v1
```

:::

## 이 포크는 무엇을 하려는가?

원본 Archon은 **AI 코딩 에이전트를 위한 워크플로 엔진**입니다. 이 포크는 그 기반 위에 한국어 개발자를 위한 문서, 랜딩, 운영 흐름, 실험 기능을 추가하는 작업 공간입니다.

## 작업 원칙

- 제품 코드와 랜딩/문서는 같은 모노레포에서 함께 관리합니다.
- 랜딩 페이지는 `packages/docs-web`에서 먼저 만듭니다.
- 제품 기능 개선은 `packages/*`에서 별도 브랜치로 분리합니다.
- 원본 추적은 유지하되, 한국어 정보 구조는 포크에서 독립적으로 다듬습니다.
