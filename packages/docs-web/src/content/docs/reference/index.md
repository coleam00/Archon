Archon 내부 구조를 다루는 기술 레퍼런스 문서입니다. HarnessLab은 Archon fork이므로 내부 패키지명과 런타임 명칭은 Archon을 유지합니다.

## 레퍼런스 문서

- **[아키텍처](/reference/architecture/)** -- 시스템 개요, 인터페이스, 데이터 흐름, 확장 가이드
- **[Archon 디렉터리](/reference/archon-directories/)** -- 디렉터리 구조, 경로 해석, 설정 시스템
- **[CLI 레퍼런스](/reference/cli/)** -- 모든 CLI 명령, 플래그, 사용 예시
- **[명령 레퍼런스](/reference/commands/)** -- 모든 플랫폼 어댑터에서 사용할 수 있는 slash command
- **[변수 레퍼런스](/reference/variables/)** -- 명령과 워크플로의 변수 치환($ARGUMENTS, $BASE_BRANCH 등)
- **[API 레퍼런스](/reference/api/)** -- 프로그래밍 방식 접근을 위한 REST API 엔드포인트
- **[데이터베이스](/reference/database/)** -- 스키마, 마이그레이션, SQLite/PostgreSQL 설정
- **[설정 레퍼런스](/reference/configuration/)** -- 전체 config.yaml 옵션, 환경 변수, 스트리밍 모드
- **[보안](/reference/security/)** -- 권한 모델, 인가, webhook 검증, 데이터 프라이버시
- **[문제 해결](/reference/troubleshooting/)** -- 자주 발생하는 문제와 해결책
