Archon 코드베이스에 기여하는 개발자를 위한 가이드입니다. HarneesLab은 Archon fork이므로, 이 섹션은 upstream과의 호환성을 유지하면서 문서와 기능을 개선할 때 기준으로 삼습니다.

## 개발자 가이드

- **[새 개발자 가이드](/contributing/new-developer-guide/)** — 코드베이스 방향 잡기, 아키텍처 개요, 첫 단계
- **[CLI 내부 구조](/contributing/cli-internals/)** — CLI 패키지 내부, 명령 라우팅, worktree 로직
- **[릴리스](/contributing/releasing/)** — 버전 관리와 릴리스 절차
- **[DX 특이사항](/contributing/dx-quirks/)** — 알려진 개발 경험 이슈와 우회 방법
