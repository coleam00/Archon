# Workflow Summary (Local)

**Local-only version** - No PR creation, no network operations.

---

## Your Mission

Summarize the completed local workflow and provide manual push/PR instructions.

1. Read and summarize all workflow artifacts
2. Show current git state
3. Prepare PR body template
4. Provide manual push/PR instructions

**Output artifact**: `$ARTIFACTS_DIR/local-summary.md`

---

## Phase 1: GATHER - Read Workflow Artifacts

### 1.1 Read Artifacts

```bash
# Check which artifacts exist
ls -la $ARTIFACTS_DIR/

# Read key artifacts
cat $ARTIFACTS_DIR/plan-context.md 2>/dev/null || echo "No plan context"
cat $ARTIFACTS_DIR/implementation.md 2>/dev/null || echo "No implementation summary"
cat $ARTIFACTS_DIR/validation.md 2>/dev/null || echo "No validation report"
cat $ARTIFACTS_DIR/review/synthesis.md 2>/dev/null || echo "No review synthesis"
```

### 1.2 Extract Key Information

From artifacts, extract:
- **Plan summary**: What was the goal?
- **Implementation**: What files changed? What was built?
- **Validation**: Did all checks pass?
- **Review findings**: What issues were found and fixed?

**PHASE_1_CHECKPOINT:**

- [ ] All artifacts read
- [ ] Key information extracted

---

## Phase 2: STATUS - Current Git State

### 2.1 Git Status

```bash
echo "## Git Status"
git status --short

echo ""
echo "## Branch Information"
CURRENT=$(git branch --show-current)
BASE=$BASE_BRANCH
echo "Current: $CURRENT"
echo "Base: $BASE"

COMMITS=$(git rev-list --count $BASE..HEAD)
echo "Commits ahead: $COMMITS"
```

### 2.2 Diff Summary

```bash
echo ""
echo "## Changes Summary"
git diff --stat $BASE..HEAD
```

### 2.3 Uncommitted Changes Check

```bash
if [ -n "$(git status --porcelain)" ]; then
  echo ""
  echo "⚠️  Warning: Uncommitted changes present"
  echo "Review with: git status"
else
  echo ""
  echo "✓ All changes committed"
fi
```

**PHASE_2_CHECKPOINT:**

- [ ] Git status retrieved
- [ ] Diff summary generated
- [ ] Uncommitted changes checked

## Manual Next Steps

### Step 1: Review Changes

```bash
# Review all changes
git diff $BASE_BRANCH

# Or review specific files
git diff $BASE_BRANCH -- path/to/file
```

### Step 2: Commit (if needed)

```bash
# Check for uncommitted changes
git status

# If present, commit them
git add {files}
git commit -m "Implement review fixes"
```

### Step 3: Push to Remote

```bash
# Push current branch to remote
git push origin HEAD

# If branch doesn't exist remotely yet
git push -u origin HEAD
```

### Step 4: Create Pull Request

**Using GitHub CLI:**

```bash
gh pr create \
  --base $BASE_BRANCH \
  --title "{feature-title}" \
  --body-file $ARTIFACTS_DIR/.pr-body.md
```

**Using GitLab CLI:**

```bash
glab mr create \
  --target-branch $BASE_BRANCH \
  --title "{feature-title}" \
  --description "$(cat $ARTIFACTS_DIR/.pr-body.md)"
```

**Manually:**

1. Visit: https://github.com/{owner}/{repo}/compare/{base}...{branch}
2. Click "Create pull request"
3. Copy content from `$ARTIFACTS_DIR/.pr-body.md`
4. Paste into PR description
5. Submit

## Alternative: Run Full Workflow

If you want automated PR creation, run the full workflow:

```bash
# For Archon workflows
archon workflow run idea-to-pr --branch $(git branch --show-current)

# For GDIT workflows
archon workflow run gdit-sdaf-idea-to-pr --branch $(git branch --show-current)
```

This will:
- Commit any remaining changes
- Push to remote
- Create PR automatically
- Run PR verification

## Troubleshooting

### Push Rejected (Branch Protection)

If push is rejected due to branch protection:

```bash
# Check protection rules
gh repo view --json branchProtectionRules

# May need to push to a different branch
git push origin HEAD:review/{branch-name}
```

### PR Already Exists

If PR already exists for this branch:

```bash
# List existing PRs
gh pr list --head $(git branch --show-current)

# Update existing PR
gh pr edit {pr-number} --body-file $ARTIFACTS_DIR/.pr-body.md
```

### Uncommitted Changes

If you have uncommitted changes:

```bash
# Review what's uncommitted
git status

# Option 1: Commit them
git add {files}
git commit -m "..."

# Option 2: Stash them
git stash

# Option 3: Discard them (careful!)
git checkout .
```
