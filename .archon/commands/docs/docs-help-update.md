# Documentation Help Update

**Input**: $ARGUMENTS

---

## Mission

Draft updates to help docs or user guides for a changed feature.

## Process

1. Read PRD, design, implementation, QA, and docs impact artifacts.
2. Identify changed user tasks and required screenshots or examples.
3. Write `$ARTIFACTS_DIR/docs/user-guide-delta.md`.

## Artifact Format

```markdown
# User Guide Delta

## Audience

## Changed User Task

## Draft Copy

## Screenshots Or Assets Needed

## Review Notes
```

## Output

Return the artifact path.
